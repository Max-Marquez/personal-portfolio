This is :Rock Paper Scissors

Made using HTML, CSS, JavaScript

This is a Rock Paper Scissor game played against a computer. The first player to 5 points wins! Here's how it works:

We use JS to listen for a mouse click event on each button for the players choice. Once the event is activated, we use JS to generate a random choice for the CPU. Once we have both players choices, we use JS to evaluate which player wins that round. After each round we check to see if either play has reached 5 points. Once a player has reached 5 points the game is over and a message is displayed declaring a winner.
