This is :Local Weather App

Made using HTML, CSS, JavaScript

This is a Local Weather App that gains the users geolocation and then uses an API to get their local weather! There is also a button that will switch between Fahrenheit and Celsius temps in case you're curious. Here's how it works:

We use JS and an API to get the users location. Once we have the geolocation, the info is put into the API and the local weather data is provided to the console. We use JS and CSS to display the relevant data to the user. We change the weather icon and background color depending on what the weather's like and the time of day.

* Note: If you decide to try out this app it will use your devices geolocation in order to find the local weather.
